# moonforce

A MoonBit port of [d3-force](https://github.com/d3/d3-force) — force-directed graph layout using velocity Verlet integration.

## About

**moonforce** brings the classic D3.js force simulation to the MoonBit ecosystem. It simulates physical forces on graph nodes to produce aesthetic, readable network layouts. Use it for network visualization, hierarchy exploration, bubble charts, and any graph that needs automated layout.
